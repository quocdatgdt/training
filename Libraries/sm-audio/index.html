$('.sound-block').smAudio({});
Depend on jquery 1.7+

  •  controlSectionSelector : controls section selector String
    Default:
    '.list-control'
  •  songSectionSelector --> songs section selector. String
    Default:
    '.list-songs'
  •  currentSong --> display index of song. int
    Default:
    0
  •  activeClass --> display class active. String
    Default:
    'active'
  •  hasVisible --> show/ hide audio. Boolean
    Default:
    true
  •  audioAttributes --> add attributes of audio. Ex: controls, size, preload,... Object
    Default:
    {controls: 'controls'}
  •  dataControlAttrs --> get attribute controls on audio. String
    Default:
    'control'
  •  dataSourceAttrs --> get attribute source on audio. String
    Default:
    'src'
  •  onCreate --> happen when create audio
  •  onPlay --> happen when audio play
  •  onPause --> happen when audio pause
  •  onError --> happen when audio error
  •  onTimeupdate --> happen when time update
  •  onEnded --> happen when audio ended
  •  onAbort --> happen when audio abort
  •  onLoadeddata --> happen when audio load data
  •  onChange --> happen when change source
  •  onWarning --> happen when browser is not supported html5 audio
  •  updateSource --> update source audio for audio
    @params:
    sources - String: source audio
  •  setCurrentTime --> set current time audio
    @params:
    time - String: time need set
  •  change --> change song.
    @params:
    song - String/ int : object element or index song
